## 概要

記事公開時にXへ投稿するための文案（テキスト＋ハッシュタグ）を自動生成するCLIスクリプト。
**このフェーズではX APIには接続しない。文面をstdoutに出力するだけ。**
人間がstdoutの出力をコピーしてXに手動投稿する運用。
X API接続はフェーズ⑤（チケット007）で行う。

## 仕様詳細

### 処理フロー

```
1. --title/--url/--category または --post-id から記事情報を取得
2. 必須ハッシュタグ（#巨人 #ジャイアンツ）を設定
3. カテゴリ別ハッシュタグを付与
4. タイトルから選手名を検出して選手別ハッシュタグを付与
5. 「タイトル\nURL\nハッシュタグ」の形式で組み立て
6. 140字超の場合はタイトルを切り詰め
7. stdoutに出力
```

### ハッシュタグルール

```
必須：#巨人 #ジャイアンツ
任意：#プロ野球 #NPB #セリーグ
選手別：記事タイトルから選手名を検出して自動付与
```

### カテゴリ別ハッシュタグ

| カテゴリ | 追加タグ |
|---------|---------|
| 試合速報 | `#プロ野球` `#セリーグ` |
| 選手情報 | `#プロ野球` |
| 首脳陣 | `#プロ野球` |
| ドラフト・育成 | `#ドラフト` |
| OB・解説者 | `#プロ野球OB` |
| 補強・移籍 | `#FA` |
| 球団情報 | `#東京ドーム` |
| コラム | `#プロ野球` |

### 出力例

```
巨人、広島に3-2で勝利！ウィットリーが7回2失点の好投
https://yoshilover.com/archives/12345
#巨人 #ジャイアンツ #プロ野球 #セリーグ
```

### CLIオプション一覧

| オプション | 説明 |
|-----------|------|
| `--title` | 記事タイトル |
| `--url` | 記事URL |
| `--category` | カテゴリ名 |
| `--post-id` | WPの投稿ID（これ単体でも動く） |
