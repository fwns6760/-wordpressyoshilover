## GCPインフラ概要

- **Cloud Run**: `yoshilover-fetcher` でRSS取得〜WP下書き生成を実行
- **Cloud Scheduler**: 1日4回（07:00/12:00/17:00/22:00 JST）自動発火 + fact check通知job
- **Secret Manager**: WP認証、X API、Gmail appパスワードを管理
- **Artifact Registry**: Dockerイメージ保管
- **GitHub Actions**: push時にDockerビルド→Artifact Registry→Cloud Run deployまで自動

## のもとけベンチマーク分析

- **dnomotoke.com**: 中日ドラゴンズまとめ。2〜3人チームで運営
- **真のSEO戦略**: 巨大サイトではなく選別的インデックスが強い可能性
  - のもとけは1,642ページ（2026-04-17調査時点）
  - 巨人系競合は5,000〜30,000ページ規模
  - のもとけの選別的インデックスが高品質密度を実現していると仮説
- **ニッチ深掘り**: 2軍・故障・首脳陣・公示など競合が拾わないカテゴリを丁寧に
- **Xポスト埋め込みが核コンテンツ**: 公式・マスコミXを引用することで一次情報性を担保

## 戦略ポジショニング

- **マスコミ棲み分け**: スポーツ紙まとめ（B型記事）とマスコミ/公式X引用（B.5型）を分離
- **のもとけモデル**: Xポスト埋め込みを記事の核に置き、ファン向け解説・視点を付加
- **汎用まとめサイトとの差別化**: 巨人特化、選手・首脳陣ニッチ深掘り、AI補助継続
