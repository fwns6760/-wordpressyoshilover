# Yoshihiroさんのコミュニケーションスタイル

## 基本的な口調パターン

### 簡潔型（最も多い）
- 「したい」「やる」「OK」「なるほど」
- 判断は短文。長い説明はしない
- 例: 「それでいい」「じゃあそれで」「やってみる」

### 疑問型（違和感の表明）
- 「これでいいの？」「〜なんでは？」「どういうこと？」
- = 「ちょっと待って、確認させて」のサイン
- 攻撃ではなく、理解を求めている
- 対応: 1〜2文で端的に説明する。長い説明は逆効果

### 指摘型（修正要求）
- 「〜じゃなくて」「〜のほうがいい」「〜はやめて」
- 具体的な変更要求。即座に受け入れて実行する
- 「なぜそうしたか」の言い訳は不要

### 満足型
- 「いいね」「完璧」「助かる」「これが欲しかった」
- → 記憶しておくべきパターン（次も同じようにする）

## 疲れのサイン

- タイプミスが増える（「泊まった」→「撮った」等）
- 文が短くなる、返答が遅くなる
- 「調子…」「ちょっと待って」「今日はここまで」

**疲れサインが出たら**: 「今日はここまでにしましょう」と提案する。作業を続けない。
