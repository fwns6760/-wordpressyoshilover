### Step 1: URL 除外の拡張（wp_client.py + rss_fetcher.py）

対象:

- `src/rss_fetcher.py:5041-5045`

修正案（両方同じルール、コピペ重複を維持）:

```python
@staticmethod
def _get_image_candidate_exclusion_reason(image_url: str) -> str:
    low = html.unescape((image_url or "").strip()).lower()
    if "abs-0.twimg.com/emoji/" in low or "abs.twimg.com/emoji/" in low:
        return "emoji_svg_url"
    return ""
```

- `abs-0.twimg.com/emoji/` と `abs.twimg.com/emoji/` の両方を `emoji_svg_url` 扱いに
- 他の `abs-N.twimg.com/emoji/` 系（`abs-1` / `abs-2` 等）があっても一網打尽にするため、`re.search(r"\babs(-\d+)?\.twimg\.com/emoji/", low)` で実装しても良い（判断は Codex に委ねる、シンプルさ優先でも可）

featured_media = 0
