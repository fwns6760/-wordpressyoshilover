## 背景（要約、第19便より）

- 62584 / 62585 / 62587 が同一経路で featured_media=0
- 真因:
  1. 画像候補の先頭に `https://abs.twimg.com/emoji/v2/svg/26a0.svg`
  2. 既存の URL 除外は `abs-0.twimg.com/emoji/` のみ → `abs.twimg.com/emoji/` を取り逃がす

  4. 1件目失敗時の fallback が無い（`_article_images[1:]` へ落ちる経路なし）
- 近傍 11 件中 7 件は画像あり → global systemic ではない、social_news/X 由来の局所 systemic

def _get_image_candidate_exclusion_reason(image_url: str) -> str:
