## Step 2. T-001 WP plugin 修正

### 停止理由

依頼書では:

- `src/yoshilover-exclude-cat.php` を修正
- WP 側へ差し替え（FTP / xserver ファイルマネージャ / Git 連携）
- その後 curl 3 本で確認

が必要ですが、このセッション内で **Xserver 側へ plugin ファイルを実反映する経路情報が見つかりませんでした**。

確認したもの:

1. `~/.ssh/config`
   - GitHub 用 `github-yoshilover` のみ
   - Xserver 用 host / user 定義なし

2. `gcloud secrets list --project baseballsite`
   - `yoshilover-wp-app-password` は存在
   - ただし Xserver FTP/SFTP/SSH 接続に使える secret は見当たらず

3. repo / docs / `.env`
   - `WP_URL`, `WP_USER`, `WP_APP_PASSWORD` はある
   - しかし plugin ファイルをサーバーへ置き換えるための
     - Xserver host
     - SSH user
     - FTP/SFTP 資格情報
     - Git pull 運用手順
     は見つからず

### 判断

WordPress REST API の資格情報だけでは plugin PHP ファイルの反映はできません。  
手元にない接続経路を推測して進めると事故リスクがあるため、依頼書どおり **Step 2 で中止** しました。
