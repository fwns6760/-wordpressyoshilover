## 対応内容

- Flash 生成文に対する post-gen validate を追加
- validate 対象
  - `close_marker`: 最終 section に `気になります / 注目です / 見たいところです / と思います` のいずれかがあるか
  - `intro_echo`: 本文冒頭が `あなたは` または `読売ジャイアンツ専門ブログ` で始まっていないか
- validate 失敗時は `wp.create_post()` 前に skip するよう変更
- skip event:
  - `article_skipped_post_gen_validate`
  - payload: `event / post_id(null) / title / post_url / category / article_subtype / fail_axis`
- `flow_summary.skip_reasons` には `post_gen_validate` が加算される
