## 対応内容

- `social_news` の source fact block を RSS entry の raw material から拡張
- 追加で使う材料
  - `entry.summary`
  - `entry.description`
  - `entry.content[].value`
- `content[].value` は RSSHub 側で引用 RT や補足本文を含むケースがあるため、social 専用 material に加算
- social 用追加 material は 300 文字 cap

## 実装方針

- 追加 API 呼び出しは入れていない
- `oEmbed` 取得は本便では未採用
  - RSS entry だけで完結する方を優先
- 取得失敗時は従来どおり `title + summary` ベースにフォールバック
